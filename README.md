# STARS-V Mk II - 3D-printable replica (non-functional)

Snap-fit display replica; the three knobs stay turnable after clicking on.

**View in browser:** https://3dviewer.net/#model=https://raw.githubusercontent.com/0xStoneyStark/3d-radio-replicas/main/STARSV-MK2_ASSEMBLED.3mf

- STARSV-MK2_PLATE1_GREEN.3mf / PLATE2_BLACK / PLATE3_PANEL_2COLOUR - per-colour print plates (350x350 bed; plate 3 uses two filament swaps at Z 3.0 / 3.6 mm)
- STARSV-MK2_PRINT_multicolour.3mf - all-in-one AMS plate
- STARSV-MK2_ASSEMBLED.3mf - assembled reference model
- STARSV-MK2_assembled.glb - GLB preview
